Meteor Mapper

An Educational Data Visualisation Tool

Meteor Mapper brings a galactic size data set down to a relatable level by plotting meteorite falls in a fun and interactive user experience. Meteor Mapper was built as an educational tool bring the fun of a science museum to a self guided learning experience. Meteor Mapper retrieves data from the NASA Open Data API, uses Bootstrap and CSS3 for styling, the Google Maps Javascript Api and Google Visualization for Data display, and extensive use of JQuery to leverage dynamic data display from an otherwise static feature.

Muzica

A Music Collaboration App

Muzica harnesses the power of collaboration to help give musicians a tool to better work together. The idea of Muzica came about by talking to a stakeholder in the recording industry who expressed his dismay with how tracks are shared between collaborators, sighting the shortcomings with current practices. Muzica allows users to build profiles, start projects, add team members, and share tracks to help everyone make better music. Muzica uses Materialize for styling, leverages SoundCloud widgets for a custom API, and PostgreSQL for data storage.

Pixel Art Maker

A Pixel Pant Playground

Pixel Art Maker is a classic throw back to Microsoft paint and other early Web Paint tools. Pixel Art Maker is a fun and user friendly pixel paint canvas that lets the users imagination run wild and creativity flow. Pixel Art Maker uses CSS3 for styling and and vanilla Javascript for DOM manipulation and paint functionality.
